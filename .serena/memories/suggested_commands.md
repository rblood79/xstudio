# XStudio 개발 명령어

## 주요 개발 명령어

### 개발 서버
```bash
npm run dev
```
- Vite 개발 서버 시작
- 핫 리로드 지원

### 빌드
```bash
npm run build
```
- TypeScript 컴파일 후 Vite 프로덕션 빌드 생성

### 린팅
```bash
npm run lint
```
- ESLint를 사용한 코드 품질 검사

### 프리뷰
```bash
npm run preview
```
- 빌드된 애플리케이션 미리보기

### Storybook
```bash
npm run storybook
```
- Storybook 개발 서버 시작 (포트 6006)
- 컴포넌트 문서화 및 테스트

## 시스템 명령어 (macOS)
- `ls`: 파일 목록 조회
- `cd`: 디렉토리 이동
- `grep`: 텍스트 검색
- `find`: 파일 검색
- `git`: 버전 관리
